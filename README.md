# 📊 Task-3 Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview
This project analyzes marketing campaign data to understand funnel performance and identify drop-off points.

## 🎯 Objectives
- Calculate traffic-to-conversion rate
- Identify drop-off percentage
- Compare segment performance
- Generate business recommendations

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Metrics
- Total Contacts
- Total Conversions
- Conversion Rate
- Drop-Off Rate
- Monthly Performance

## 📈 Key Insights
- Overall conversion rate: 11.52%
- Major drop-off observed after initial contact
- Previous successful campaigns improve conversion probability.
